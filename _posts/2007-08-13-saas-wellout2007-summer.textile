株式会社ディー・ワークス(本社・東京都港区　代表取締役・泉　大五郎)は、携帯サイト専用のアクセス解析サービス「wellout 2007-summer バージョン」をリリース、本日、8月13日より提供を開始します。
　「wellout」はiモード、EZweb、Softbank、Yahoo!ケータイの公式・勝手サイトのアクセス状況をさまざまな角度から解析する、SaaS型のサービスです。
　今回のバージョンアップでは、拡大を続けるモバイルリスティング広告への対応を全面的に強化しており、これまで以上に、広告効果が詳細に検証できるよう、設計されています。
　主な機能として、①モバイルリスティング広告の出稿キーワードごとに、検索キーワード及び検索元サイトからの流入・コンバージョンがわかるようになりました　②自然来訪時のコンバージョン測定をさらに強化し、検索キーワード、リンク元サイトごとの流入・コンバージョンの測定を可能にし、その結果、モバイルSEOを行う上でも大変参考となるレポートの提供を実現　③Google AdWords API接続時にこれまで提供していた利用金額、インプレッション数、クリック数に加えて、平均掲載順位も併せて表示　④モバイルリスティング広告への対応を強化するために、効果測定レポートの構造をGoogle AdWords、Overtureのレポート形式に準じた形式に変更するとともに、効果測定全体でコンバージョンページを6個まで指定可能。この他には、任意のページ(URL)をグループ化し、ページビュー数、ユニークユーザ数を抽出・レポート化する「ページグルーピング機能」も追加されています。
　「wellout」は2005年10月のサービス開始以来、四半期に一度のペースでバージョンアップを進めており、今回で6回目となります。同サービスは小売、金融関係、求人、旅行、美容・エステ、教育など幅広い業界の携帯サイトで利用されており現在、約30サイトに導入されています。 

■wellout 2007-summer バージョンの新機能


広告効果測定機能 
出稿広告単位での検索キーワード、リンク元サイト別、コンバージョン測定 
自然来訪時のコンバージョン測定を強化 
Google AdWords API利用時に平均掲載順位の表示を追加 
新レポート形式導入に伴い成約ページを6個まで指定可能 
ページグルーピング機能 
ページ内訳、開始ページ、離脱ページ、一ページ離脱各レポートで任意のページ（URL）をグループ化 

■wellout について
　2005年10月、国内で初めて、携帯サイト専用のアクセス解析サービスをSaaS型で提供。4半期に一度のペースでバージョンアップを行っている。 

　●主な特徴
　・iモード、EZweb、Yahoo!ケータイの公式・勝手サイトに対応
　・ページビュー、訪問回数、ユニークユーザなどのアクセス動向を解析
　・広告効果測定機能で広告からの流入・コンバージョン測定に加え、自然来訪からのコンバージョンも測定
　・携帯端末の機種を世代別・機能別などで分布、表示
　・サイト内でのユーザーの動きを知る経路解析


　●対応サーバー
　・Apache
　・Microsoft IIS
　・BEA WebLogic Server
　・IBM WebSphere Application Server
　・Oracle Application Server
　・Sun Java System Web Server
