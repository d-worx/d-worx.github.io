　株式会社ディー・ワークス(本社・東京都港区　代表取締役・泉　大五郎)は、携帯サイト専用のアクセス解析・広告効果測定サービス「wellout」を株式会社NTTドコモのiモードIDに対応し、本日4月1日より提供を開始します。
　「wellout」はiモード、EZweb、Yahoo!ケータイの公式・勝手サイトのアクセス状況をさまざまな角度から解析する、SaaS型のサービスです。
　これまでiモード対応の勝手サイトにおいては、ユニークユーザを判別することができませんでした。 iモードIDに対応することで、ユニークユーザを判別することが可能となり、これまで以上に「wellout」において詳細なアクセス解析・広告効果測定が可能となります。
　「wellout」では、これまでユニークユーザを判別することができないサイトへの訪問に対しては、「ユニークユーザ不明訪問数」というレポート項目を用意することで対応してきました。 iモードIDに対応することで、以下のレポートにおいて、すべてのキャリアでユニークユーザ数を把握することが可能となります。 

■ユニークユーザが表示可能なレポート
　アクセス動向レポート
　　・日別レポート
　　・曜日別レポート
　　・時間帯別レポート
　　・訪問回数レポート
　　・ページ内訳レポート
　経路解析レポート
　　・開始ページレポート
　　・離脱ページレポート
　　・1ページ離脱ページ
　広告効果測定レポート
　端末情報レポート


■wellout について
　2005年10月、国内で初めて、携帯サイト専用のアクセス解析サービスをSaaS型で提供。小売、金融関係、求人、旅行、美容・エステ、教育など幅広い業界の携帯サイトで利用されており現在、約80サイトに導入されています。

　●主な特徴
　・iモード、EZweb、Yahoo!ケータイの公式・勝手サイトに対応
　・ページビュー、訪問回数、ユニークユーザなどのアクセス動向を解析
　・広告効果測定機能で広告からの流入・コンバージョン測定に加え、自然来訪からのコンバージョンも測定
　・携帯端末の機種を世代別・機能別などで分布、表示
　・サイト内でのユーザーの動きを知る経路解析

　●対応サーバー
　・Apache　　
　・Microsoft IIS（専用モジュールを提供）
　・BEA WebLogic Server
　・IBM WebSphere Application Server
　・Oracle Application Server
　・Sun Java System Web Server
