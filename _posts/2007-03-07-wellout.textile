株式会社ディー・ワークス(本社・東京都港区　代表取締役社長・泉　大五郎)は、携帯サイト専用のアクセス解析「wellout 2007-spring バージョン」をリリース、3月8日よりサービスを開始します。
　「wellout」はiモード、EZweb、Yahoo!ケータイの公式・勝手サイトのアクセス状況をさまざまな角度から解析する、SaaS型のサービスです。
　今回のバージョンアップでは、「IISロギングモジュール」「滞在時間レポート」「広告効果測定の強化」などの新機能を追加しています。
「IISロギングモジュール」は当社が独自に開発した、Microsoft Windows Server2003 のIIS6.0(以下、IIS 6.0)のアクセスログを出力するモジュールです。これまでIIS 6.0が出力するアクセスログでは携帯サイトを正確に解析できませんでした。「IISロギングモジュール」をインストールことにより、携帯キャリア固有のHTTPヘッダー情報を含んだ形でアクセスログが出力されます。これにより、IIS 6.0においても携帯サイトの正確なアクセス解析を実現。「滞在時間レポート」は、ユーザーのサイト内における滞在時間を解析します。「広告効果測定の強化」は、これまでの解析結果に訪問ユニークユーザ数、成約ユニークユーザ数が追加され、グラフ表示も可能となります。このほかにも、時間帯別での新規ユーザー、再訪問ユーザー数などを表示する「時間帯別レポート」などの機能も搭載されています。
　2005年10月のサービス開始以来、四半期ごとにバージョンアップを進めており、今回で5回目となります。同サービスは小売、金融関係、求人、美容・エステ、教育など幅広い業界の携帯サイトで利用されており現在、約20サイトに導入されています。コンビニエンス大手・ローソンでは、2007spring バージョンを、同社のECサイトおよびキャンペーンサイトに3月より利用することが決まっています。当社代表の泉は「クライアントの要望そして市場のニーズをきちんと押さえながら、さらに高品質、信頼性の高い『wellout』に進化させたい」と語っています。
　今後、当社では、「wellout」で蓄積した技術・ノウハウを活用し、今年末から来年に向けて新たなサービスの提供を計画しています。

■wellout について
　2005年10月、国内で初めて、携帯サイト専用のアクセス解析サービスをSaaS型で提供。4半期に一度のペースでバージョンアップを行っている。
　●主な特徴
　・iモード、EZweb、Yahoo!ケータイの公式・勝手サイトに対応
　・ページビュー、訪問回数、ユニークユーザなどのアクセス動向を解析
　・広告効果測定機能で広告からの流入・コンバージョンを測定
　・携帯端末の機種を世代別・機能別などで分布、表示
　・サイト内でのユーザーの動きを知る経路解析
　●対応サーバー
　・Apache
　・Microsoft IIS
　・BEA Weblogic Server
　・IBM WebSphere Application Server
　・Oracle Application Server
　・Sun Java System Web Server
