株式会社ディー・ワークス（本社：東京都港区、代表取締役：泉　大五郎）は、同社の携帯サイト専用アクセス解析・効果測定サービス「wellout」に、GPS（位置情報）解析機能などを追加した“2010-spring”バージョンを3月1日より提供開始します。

●wellout ( "http://www.wellout.net/":http://www.wellout.net/ )

■バージョンアップ内容
(1)「GPS(位置情報)解析レポート」

　携帯サイト専用アクセス解析として初めて、ユーザーがどの場所から自サイトにアクセスしているか、最小500m四方単位の地図上にて分析することができます。GPS解析レポートはベータ版としての公開であり、今後個々のお客様の要望に応じ、カスタマイズを行った上で提供します。

　昨今、GPS機能を活用した周辺店舗検索やクーポン・キャンペーンなどの情報取得、ソーシャルゲームや交通ナビゲーションなど様々なサービスが普及してきています。また、welloutで解析しているお客様サイトの全ユーザーの8割がGPS対応の機種を利用している状況です。交通広告や屋外広告などの OOHメディアにおいては、その場で情報を伝えるだけでなく、モバイルサイトへの誘導が目的の一つにされつつあります。

　GPS解析レポートは、エリアマーケティングを行う際の仮説と検証に活用可能です。ユーザーの位置情報と店舗やOOHメディアの位置とをかけ合わせて解析することで、店舗誘導のためのプロモーションの効果をアクセスエリアの側面から把握できます。

"GPS(位置情報)解析レポートについてはこちら\":http://www.wellout.net/report/gps.html

(2)「広告からの経路解析機能」

　モバイル広告の効果解析をより深く行いたいというお客様の声を反映し、携帯サイト専用アクセス解析として初めて、広告からの経路解析に対応しました。「広告から流入したユーザーが、どのページで迷っているか」、「コンバージョンページに流入したユーザーの、一番多い導線はどれか」等、これまで把握のできなかった、広告毎のユーザーの動きを分析できるようになりました。

"広告からの経路解析機能についてはこちら\":http://www.wellout.net/report/adc.html#path

(3)「IIS7.0、7.5へ対応」

　welloutのアクセスログ版では、独自に開発したIISロギングモジュールを用いることでMicrosoft IISサーバの解析を行うことができます。今回このIISロギングモジュールをバージョンアップし、これまでのIIS6.0に加え、Windows Server 2008(32bit・64bit)、Windows Server 2008R2(64bit)のIIS7.0、7.5にも対応可能となりました。

"IISロギングモジュールについてはこちら\":http://www.wellout.net/aboout/system.html#iis

この他にも、お客様から要望受け、レポート画面にイベントやコメントを記述できるメモ機能、見たいページの解析結果のみを抽出できる検索・フィルタリング機能などの新機能に加え、レポート内の約70項目について見直しを図りました。welloutは今後もお客様の声を反映したアップデートをおこなっていく予定です。

■wellout について
2005年10月世界初の携帯サイト専用のアクセス解析サービスをSaaS型で提供。年3回のペースでバージョンアップを行っています。
小売、保険・金融、各種メーカー(自動車、化粧品、ビール等)、求人、旅行、美容、出版、教育など、幅広い業種に約250サイトへの導入実績があります。

●主な特長
・アクセスログ版、ビーコン版と2種類の計測手法が選択でき、幅広い環境にて導入が可能(ホスティングでの導入も可)。
・iモード、EZweb、Yahoo!ケータイの公式・勝手サイト及び、iPhone、Androidに代表されるスマートフォンに対応。
・広告効果測定機能で広告からの流入・直接コンバージョン測定に加え、間接効果の計測も可能。また自然来訪からの測定も可能。
・サイト内のユーザーの動きを多方面から分析できるFlashベースの動的経路解析。
・約950機種の端末データを基に機能別などで分布、表示するアクセス分析レポート。

●対応サーバー
・Apache、Microsoft IIS等のWebサーバー(ホスティング可)
